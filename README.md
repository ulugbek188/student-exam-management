#  Go Day 02 - Student Exam Management System

A simple console-based Student Exam Management System built with Go.

This project is part of my **30-Day Golang Challenge**, where I build one mini project every day to improve my Go programming skills.

---

##  Project Description

This application manages a student's exam information and generates a complete academic report.

The program demonstrates fundamental Go programming concepts such as variables, constants, functions, conditional statements, loops, and multiple return values.

---

##  Features

- Display student information
- Display exam information
- Calculate student grade
- Check pass or fail status
- Evaluate student performance
- Determine scholarship eligibility
- Display current subjects
- Generate a complete student report
